
  # potential-enigma


  

  ## Description

  ### This application creates a README.md file easily using a pre-built template. The user is prompted with a series of questions and the application inserts these answers within the template.

  ### Inquirer, Node.js

  ## Usage <a id="usage"></a>

  ### The user needs to clone the repo containing the README.md generator (https://github.com/gwacky/readme-generator). Inquirer is required, npm install on the command line to download the necessary files. Once installed, change the directory to the develop folder within potential-enigma. On the command line, run node index.js and answer prompts. README.md file will be loaded into folder.
